Projects

QuickCNN

Created high-level library in Python which is backed by the Keras, TensorFlow, and Scikit-learn to exercise faster experimentation with Convolutional Neural Networks(CNN) in Google-Colaboratory. It is intended to quickly play with the ConvNet architectures for transfer-learning, finetuning, scratch training, and linear classifier for layer activations.

ConvNet-Vis

Implemented an interactive tool for the visualization of 13 pretrained Deep Convolutional Neural Networks and other custom architectures. It has methods for deconvolution, for visualizing the activation of ReLU, pooling, and convolution layers, and hallucinating the image with deep-dream.

Word Embedding(N-gram)

Training noisy words from short sentences even with spelling errors through skip gram model and learn N-gram embedding for each words. We've used these word embeddings in Deep Learning NLP models. This appoarch gives better results than vanilla word2vec models.

View training

ConvNet-Zoo

Built a playground for 13 pretrained Deep Convolutional Neural Networks, which is trained in ImageNet classes to understand each layer work. It also shows the neuron excitation in each layers activation volume per channel to describe the number of ways layers looks at an image and allows you to compare the variations.

See Project
